# Chicken Party: The Last Promise

**Chicken Party** is a survival management simulation game where you manage a startup company. Your goal is to survive as long as possible by balancing profitability and employee happiness to keep your founding partners satisfied.

## Documentation

Detailed design documentation can be found in the `docs` directory:

*   [Game Design & Core Mechanics](docs/GameDesign.md)
*   [Unit & Policy Design](docs/UnitDesign.md)

## How to Play

1.  Open `index.html` in a modern web browser.
2.  **Build** units to generate R&D and Revenue.
3.  **Manage** employee happiness to prevent them from turning into Zombies.
4.  **Survive** the annual "Chicken Party" where partners evaluate your performance.

## Controls

*   **Left Click**: Select / Build / Interact
*   **Right Click** (or UI Button): Cancel / Deselect
