# Chicken Party 遊戲

這是一款名為 "Chicken Party" 的網頁遊戲。該遊戲透過模擬一個派對的情境，來展示每年參與派對的人數以及可能發生的事件。本遊戲使用 HTML, CSS, 和 JavaScript 實現，提供了一個互動式的用戶界面和基於機率的遊戲邏輯。

## 安裝

要開始使用，你只需將包含 `chicken_party.html` 文件的項目下載到你的電腦上，然後在瀏覽器中打開此 HTML 文件即可。無需進行傳統的安裝過程。

### 下載

你可以使用 Git 來克隆這個項目：

```bash
git clone [你的項目Git倉庫URL]
```



## 使用方法

要開始遊戲，請遵循以下步驟：

1. 打開 `chicken_party.html` 文件於你的瀏覽器。
2. 在適當的文本框中輸入參加人數和開始年份。
3. 點擊 "開始遊戲" 按鈕。
4. 觀看遊戲結果顯示在頁面上。

## 功能

- **用戶輸入**: 允許用戶自定義參加人數和開始年份。
- **動態結果更新**: 遊戲進行時，結果會動態更新，包括每年的參加人數和派對的狀態。
